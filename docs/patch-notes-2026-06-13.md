## 比赛与战术

### 定位球系统重构

- 角球、任意球、点球不再「触发即进球」，改为 **机会 → 发球质量 → 终结** 三阶段检定。
- 战术页可指定 **点球 / 任意球 / 角球** 主罚人（未指定则按能力自动推断）。
- 直播与赛后报告会区分定位球类型，并显示主罚球员。
- 定位球进球频率已校准至更合理的水平（约每队每场 0.3 球左右）。

### 淘汰赛加时与点球

- **欧冠决赛、世界杯淘汰赛** 等：90 分钟平局后会进入 **加时**，仍平则 **点球大战**，点球胜者正确晋级。
- **欧冠两回合** 总比分平局时，次回合也会走加时 + 点球流程。
- 世界杯 AI 模拟与玩家现场比赛 **共用同一套比赛引擎**，不再使用简化随机比分。

### 换人界面替补席显示体力

- 比赛直播 **换人面板** 中，替补球员现在与场上球员一样显示 **体力条与百分比**（绿 / 黄 / 红），便于判断该换谁。

### 换人后终局阵容与统计

- 比赛中完成的换人，赛后 **阵容、评分、出场统计** 会与终场名单一致，不再仍显示赛前 XI。

### 比赛解说与卡牌原因

- 大幅扩充比赛事件文案（进球、扑救、点球、定位球等）。
- **黄牌 / 红牌** 会附带原因（鲁莽铲球、战术犯规、破坏得分机会等）。
- 定位球、点球相关解说与标签更清晰。

## 伤病系统

### 9 种伤病类型与严重度

- 伤病不再只是「受伤 N 天」，而是有具体种类（肌肉拉伤、脚踝扭伤、腿筋拉伤等）和 **轻 / 中 / 重** 分级。
- 比赛、训练、随机事件中的伤病 **统一规则**，天数与类型一致，不再出现「跖骨骨折 3 天」这类矛盾。

### 赛季报销（极低概率）

- 极少数情况下可能出现 **ACL、跟腱断裂、重大手术** 等长期伤病，恢复至赛季结束或至少 180 天。
- 赛季报销球员会收到高优先级通知，详情页标注 **🚫 赛季报销**。
- 伤员无法挂牌转会 / 外租；国家队大名单不会新选赛季报销球员。

## 转会与市场

### 租借系统完善

- **挂牌外租**：阵容长按或球员详情合同区可操作；与出售挂牌互斥。
- **租借市场**：转会市场新增 **租借** 页，可浏览 AI 与对手挂牌的外租球员。
- **完整租借流程**（对齐永久转会）：
  1. 提交 **租借费报价** 与 **工资分担比例**（50% / 75% / 100%）
  2. 与出租方 **多轮谈判** 租借费
  3. 球员 **考虑是否外租**
  4. 协商 **租借条款**（工资分担等）
  5. 经理 **确认租入**
- 进度在 **转会市场 → 报价** 页跟踪（带 `[租借]` 标记）；重要节点会进 **收件箱**。
- **租入 / 租出管理**：查看剩余租期、**召回**、**续租**；外租球员训练页会显示出场配额与效率。
- **AI 外租挂牌**：转会窗内 AI 会按「年轻、有潜力、缺出场」画像挂牌，窗期结束自动下架。
- **修复**：租借页报价按钮无法点击；租入球员误显示「挂牌出售」；租入完成后清除错误的出售标记。

### 同一转会窗每名球员仅可流动一次

- 在 **同一转会窗**（夏窗 6–8 月 / 冬窗 1 月）内，每名球员只能完成 **一次** 流动：**永久转会、租借、自由签约** 均计入。
- 仅挂牌或谈判中不算；**租借到期归队、提前召回** 不算第二次流动。
- 下一窗开启后可再次参与转会。

### 死敌关系

- 开档初始化 **18 对经典死敌**（如曼联—利物浦、皇马—巴萨等）。
- 死敌之间转会 **意愿更低、难度更高**；德比战 **上座率提升**；转会 UI 会标注「死敌」。

### 青训球员可转会

- **青训队已签入** 的球员可以 **挂牌出售**，走正式转会报价流程（与一线队相同）。
- **青训候选名单** 中的未签入球员 **不可** 转会。
- **修复**：妖人转会后，原球队月度青训逻辑不再 **覆盖还原** 其合同与归属。

### 时间推进待办提醒（重要）

推进日期时，若有 **必须你亲自确认** 的事项，游戏会 **阻断并提示**，不能误点「下一天」跳过：

| 待办 | 跳转 |
|------|------|
| 出售待确认 | 转会市场 · 出售 |
| 买入 / 租借待确认签入 | 转会市场 · 报价 |
| 自由球员待确认签约 | 转会市场 · 自由 |
| 董事会续约待回复 | 经理资料 |
| 失业且有邀约 | 经理资料 · 求职 |
| 国家队主帅邀请 | 收件箱 |
| 世界杯大名单未锁定（开赛前 7 天内） | 国家队大名单 |

- **5 天未确认** 的买入 / 出售 / 自由签约会自动作废，阻断随之消失。
- 旧逻辑「收件箱堆满高优先级消息就阻断」已移除，改为上述 **真实业务状态** 驱动。

## 阵容与存档

### 阵容设定不再丢档

- **阵型、变体、首发 XI 槽位、替补、球员职责** 等会正确写入存档。
- 修复了首发 XI 槽位在读档时被错误「压实」导致 **球员错位** 的问题。

### 两套阵容方案（A / B）

- 阵容页顶部可切换 **方案 A / 方案 B**，分别保存不同的阵型 + 首发 + 替补。
- 支持 **保存方案**；切换时若有未保存修改会提示。
- 两套方案 **随存档持久化**；旧存档首次打开会以当前阵容初始化 A、B（内容相同）。

## 成长与训练

### 22 岁以上：出场挂钩训练效率

- **22 岁及以上** 球员的成长与 **俱乐部正式比赛出场** 挂钩：
  - 22 岁起需约 **12 场 / 赛季** 练满效率，每长一岁配额 +2，27 岁起封顶 **25 场**。
  - 无出场时训练效率约 **25%**；捞到足够出场可至 **100%**。
- **外租期间在效力俱乐部的出场同样计入**。
- **21 岁及以下** 无出场门槛，靠训练 + 青训加成即可。
- 训练页、球员详情会显示 **出场 X / 配额** 与 **训练效率 Y%**。

### 低潜力球员成长上限

- 修复低潜力（PA）球员仍可能涨到 **90+ 总评** 的问题；成长上限与潜力档一致。

### 关于「一线队训练太慢」

- 经量化测试，**非 Bug**：受每周仅 3 个训练日、接近潜力上限衰减、出场不足等因素影响，成年球员成长本就有意偏慢。22+ 需通过比赛锻炼是 **设计意图**。

## 联赛、赛季与俱乐部

### 联赛模拟更贴近实力

- 修复 **保级队长期夺冠、豪门频繁降级** 的极端随机问题。
- 强队主场胜率、弱队爆冷概率已重新校准，长期排名与球队实力 **大体一致**。

### 赛季总结页数据修正

- **联赛奖金** 的排名与金额，现在与刚结束赛季的 **真实排名** 一致（不再误显示新赛季积分榜第 1 名）。
- **财务总结** 显示该赛季真实收支（含联赛奖金），不再读档后归零。

### 顶级联赛降级 → 强制下课

- 玩家执教球队若在 **顶级联赛** 降级，将 **自动失业**（履历标注「降级解约」），需重新求职。
- 原球队由 AI 教练 **立即接管**；当季不可回聘原队。
- **次级储备池** 降级不触发此规则（当前无玩家执教次级联赛路径）。

### 董事会目标

**月度目标**

- 6 种月度目标（连胜、零封、进球、射手等）现按 **当月真实比赛数据** 判定，不再随机糊弄。
- 达成：董事会满意度 **+5**；失败：**-3**，均会通知。

**赛季目标**

- 目标档位综合 **声望 + 联赛内排名**，中游队不再被塞「联赛冠军」类离谱目标。
- **升班马** 首赛季目标上限为保级档。
- 你在赛季初确认的目标，与董事会评估、解雇逻辑 **文案一致**。
