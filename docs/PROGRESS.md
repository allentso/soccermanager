## 已验证的核心玩法循环

```
新游戏 → 创建经理 → 选择球队 → 进入仪表盘
  ↓
查看阵容 → 调整战术 → 设置训练 → 管理转会
  ↓
推进日期 → [比赛日] → 赛前确认 → 观看比赛 → 赛后结果
         → [非比赛日] → 训练/球探/青训/谈判/随机事件
  ↓
赛季结束 → 结算(奖项/升降级/奖金) → 进入新赛季
  ↓
存档 / 读档 / 继续
```
